Search Results Not Updating After Applying Filters

Bug ID BR-002
Application Demo E-commerce Website
Module Search
Severity Medium
Priority Medium
Status Open
Environment Android 15 • Google Chrome (Latest)

Summary

Search results do not refresh after a user applies one or more search filters. The displayed products remain the same even though the selected filters should narrow the results.

Preconditions

  • User is on the search results page.
  • Multiple products are available.

Steps to Reproduce

  1. Open the application.
  2. Search for "Shoes".
  3. Apply the Brand filter.
  4. Apply the Price Range filter.
  5. Observe the search results.

Expected Result

The product list should refresh immediately and display only items matching the selected filters.

Actual Result

The filters appear to be applied, but the product list does not update. Products that do not match the selected filters continue to be displayed.
