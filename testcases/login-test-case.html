Test Case TC-001

Title: Verify successful user login

Objective: Ensure users can log in with valid credentials.

Precondition: User account already exists.

Test Steps

  1. Open the login page.
  2. Enter a valid email address.
  3. Enter a valid password.
  4. Click the Login button.

Expected Result

The user is redirected to the dashboard successfully.
